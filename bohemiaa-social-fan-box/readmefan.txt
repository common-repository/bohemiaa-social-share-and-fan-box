== Installation ==

1. Upload the entire bohemiaa-social-fan-box folder to your wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Visit http://www.bohemiaa.com/index.php?L=blogs.browse to create a new Blog in Bohemiaa.
   
4. Activate the widget or use this inside your template ->  `<?php bohemiaa_fan_box('PROFILE_ID'); ?>`.

4. Go to 'Settings->Bohemiaa Fan Box' in your admin interface to select you options.
